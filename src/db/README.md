# `src/db` — Data layer (Drizzle ORM + Neon Postgres)

Populated in **Phase 3**. Planned contents:

- `index.ts` — Drizzle client bound to `env.DATABASE_URL` (Neon serverless driver).
- `schema/` — table definitions per domain (clubs, stock, sales, expenses,
  kitchen, credit, suppliers, staff, bookings).
- `seed.ts` — loads the prototype dataset (`black stars html/data.js`) for dev.

Migrations are generated by `drizzle-kit` into `/drizzle` at the repo root.
